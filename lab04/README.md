# Отчет по лабораторной работе: Анализ датчиков случайных чисел

## 1. Цель работы
Реализация базового датчика случайных чисел, сравнение его характеристик со встроенным генератором Python и теоретическими значениями для равномерного распределения на интервале [0, 1].

## 2. Реализованные алгоритмы

### 2.1 Базовый датчик (LCG - Linear Congruential Generator)
- **Параметры генератора:**
  - Модуль: $m = 2^{31}$
  - Множитель: $a = 1103515245$
  - Приращение: $c = 12345$
  - Начальное состояние: $seed = 52$
- **Формула:** $X_{n+1} = (a \cdot X_n + c) \mod m$
- **Выходное значение:** $U_n = X_n / m \in [0, 1]$

### 2.2 Встроенный генератор Python
- Используется функция `random.random()`, реализующая алгоритм **Mersenne Twister (MT19937)**.

## 3. Методология исследования
- **Объем выборки:** $N = 100\,000$ значений для каждого генератора.
- **Оцениваемые характеристики:**
  - Выборочное среднее: $\bar{x} = \frac{1}{N}\sum_{i=1}^{N}x_i$
  - Выборочная дисперсия: $s^2 = \frac{1}{N-1}\sum_{i=1}^{N}(x_i - \bar{x})^2$
- **Теоретические значения** для равномерного распределения $U[0,1]$:
  - Математическое ожидание: $\mu = 0.5$
  - Дисперсия: $\sigma^2 = \frac{1}{12} \approx 0.0833333$

1. **Реализованный LCG-генератор** с параметрами (a=1103515245, c=12345, m=2³¹) успешно генерирует последовательность, статистические характеристики которой хорошо согласуются с теоретическими ожиданиями для равномерного распределения.

2. **Оба генератора** удовлетворяют требованиям к базовым датчикам случайных чисел для большинства практических задач, что подтверждается:
   - Малыми отклонениями выборочных моментов от теоретических значений (менее 0.06%).
   - Визуальной равномерностью гистограмм распределений.
